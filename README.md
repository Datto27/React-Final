# Quiz app
- Quiz application for final project
- Used opentdb.com api for data

# Used Tools
- React v18
- axios for api requests
- scss for styles
- framer-motion for animations
- formik + yup for form handling

# Start application
```bash
/> npm i
/> npm start
```

# Structure
* Route - index.js
* Pages - /pages
* Components - /components
* Custom hooks - /hooks
* Api Handling - /api
